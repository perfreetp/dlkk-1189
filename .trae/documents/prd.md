## 1. 产品概述

SQL 教学与练习桌面客户端，面向培训机构讲师和初学数据人员进行本地化 SQL 练习。提供从课程浏览、在线编写 SQL、实时运行校验到讲师批改和错题回顾的完整学习闭环，帮助学员高效掌握 SQL 技能，帮助讲师轻松管理教学与考核。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 学员 | 本地注册/登录 | 浏览课程、练习题目、查看结果、错题本、收藏写法 |
| 讲师 | 本地注册/登录 | 创建题目、设置参考结果、查看提交记录、批量点评、导出成绩单、知识点统计 |

### 2.2 功能模块

1. **课程目录**：课程分类浏览、知识点标签、练习集选择、学习进度追踪
2. **题目练习**：SQL 编辑器、题目说明、示例数据运行、表结构查看、错误提示、标准答案对比
3. **结果校验**：查询结果展示、与参考结果比对、差异高亮、通过/未通过判定
4. **讲师批改**：提交记录查看、代码审阅、批量点评、评分、导出成绩单
5. **错题本**：错题列表、重做功能、收藏常用写法、知识点归类、掌握度追踪

## 3. 核心流程

### 学员练习流程
学员登录 → 浏览课程目录 → 选择练习集 → 进入题目 → 阅读题目说明 → 查看表结构和示例数据 → 在编辑器中编写 SQL → 执行并查看结果 → 系统自动校验 → 通过则进入下一题 / 未通过则查看错误提示 → 可查看标准答案对比 → 错题自动收录到错题本 → 在错题本中重做巩固

### 讲师管理流程
讲师登录 → 进入管理后台 → 创建/编辑题目（设置描述、参考 SQL、参考结果、知识点标签） → 查看学员提交记录 → 批量点评打分 → 导出成绩单 → 查看知识点统计通过率 → 调整教学策略

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：深色主题为主，工业实用风格（Industrial Utilitarian），搭配明亮 accent 色彩，体现数据/代码工具的专业感
- **主色调**：深灰底色（#1a1a2e）+ 深蓝辅助（#16213e）+ 青绿强调色（#0f3460 → #00d2ff 渐变）+ 橙色告警/错误（#e94560）
- **按钮风格**：圆角矩形（8px），微凸起 3D 效果，hover 时发光
- **字体**：代码区域使用 JetBrains Mono，UI 文字使用 Noto Sans SC，标题使用 Outfit
- **布局风格**：左侧导航栏 + 主内容区，题目练习页为三栏布局（题目说明 | SQL编辑器 | 结果面板）
- **图标风格**：线性图标，2px 描边，与深色主题搭配

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 登录页 | 用户认证 | 居中卡片式登录表单，角色切换 Tab，深色渐变背景，代码雨动效 |
| 课程目录 | 课程分类卡片 | 网格卡片布局，每张卡片含知识点标签（彩色 pill）、进度环形图、难度星级 |
| 课程目录 | 练习集列表 | 左侧分类树 + 右侧列表视图，列表项含完成状态图标、题目数量 badge |
| 题目练习 | 整体布局 | 三栏分栏布局，可拖拽调整宽度，顶部面包屑导航 |
| 题目练习 | 题目说明面板 | 左侧窄栏，Markdown 渲染题目描述，底部知识点标签 |
| 题目练习 | SQL 编辑器 | 中间主栏，深色代码编辑器，顶部工具栏（运行/格式化/清空），行号+语法高亮 |
| 题目练习 | 结果面板 | 右侧栏，Tab 切换（结果/表结构/示例数据/标准答案），结果表格支持排序 |
| 结果校验 | 结果对比 | 双栏并排表格，差异行用红色/绿色背景高亮，顶部通过状态大图标 |
| 讲师批改 | 题目管理 | 左侧题目列表 + 右侧编辑表单，SQL 预览区域，参考结果配置 |
| 讲师批改 | 提交记录 | 数据表格，支持筛选排序，行展开查看 SQL 详情，批量选择 checkbox |
| 讲师批改 | 知识点统计 | 柱状图/雷达图展示各知识点通过率，颜色编码（红/黄/绿） |
| 错题本 | 错题列表 | 卡片式列表，每张卡片含知识点标签、错误类型 badge、上次提交时间 |
| 错题本 | 掌握度追踪 | 雷达图展示各知识点掌握度，中心区域显示总体评分 |

### 4.3 响应式设计

- 桌面优先设计，最小支持 1280px 宽度
- 题目练习三栏布局在窄屏下可折叠左右面板
- 表格在小屏下支持横向滚动
- 移动端暂不做专门适配，主要面向桌面使用场景

### 4.4 动效设计

- 页面切换：淡入淡出 + 微位移
- 卡片 hover：上浮 + 边框发光
- 执行 SQL：加载时编辑器顶部进度条动画
- 结果出现：表格行逐行淡入
- 错误提示：抖动 + 红色闪烁边框
- 通过判定：绿色对勾缩放弹出动画
